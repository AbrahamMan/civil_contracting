Civil Contracting: User Docs

Civil Contracting is Frappe app for contractors. It is to be used along with ERPNext. It extends ERPNext for business dealing with labour directly and keep track of their outstanding wages. This App cannot be used separately and only be used along with ERPNext.

Civil Contracting creates a desk icon for the module. In this module, there are five types of Documents and one Setup section.

Contents

  1. Measurement Book
  2. Measurement Sheet
  3. Wage Slip
  4. Worker
  5. Worker Sheet
  6. Setup
